
# Voyager

## Main Features

### 1. Weather Dashboard
Displays the current weather of the selected location:
- Temperature
- Weather Condition
- Humidity
- Wind Speed

### 2. Weather Forecast
Displays the weather forecast for the next **7 days**.

### 3. Location Search
Allows users to search for a city or municipality and view its weather information.

### 4. Travel Advisory
Based on the weather, the system provides a simple advisory such as:
- 🟢 Safe to Travel
- 🟡 Travel with Caution
- 🔴 Travel Not Recommended

### 5. Weather Alerts
Displays alerts when severe weather conditions are detected, such as:
- Heavy Rain
- Strong Winds
