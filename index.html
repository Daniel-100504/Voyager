      About Voyager

      Your intelligent travel companion, combining real-time weather data with travel safety insights and ferry scheduling.

      ✨ Features

      • ⛅
        Weather Dashboard

        Live temperature, conditions, humidity, and wind speed for your destination.

      • 📅
        7-Day Forecast

        Daily highs and lows with detailed hourly temperature trends.

      • 🌐
        Global Search

        Find any city or municipality worldwide instantly.

      • 🚦
        Travel Advisory

        Automated safety ratings based on current weather conditions.

      • 🚢
        Ferry Routes

        Dynamic ferry schedules with real-time distance calculations.

      • ⚠️
        Alert System

        Instant notifications for severe weather conditions.

      📊 Data Sources

      Weather Data

      Powered by meteoblue, a leading weather API providing accurate forecasts and current conditions.

      Location Search

      Using Open-Meteo's free geocoding service for global location data.

      Ferry Routes

      Dynamically generated based on geographic proximity using Haversine distance calculations.
